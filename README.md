Цей репозиторій містить вихідний код, інструкції та результати виконання лабораторних робіт з дисципліни "Безпека програм та даних" 

Проект організовано для зручного відстеження прогресу та зберігання всіх необхідних матеріалів.

Структура Репозиторію:
Репозиторій складається з окремих папок для кожної лабораторної роботи.
Кожна папка (labXX) містить власний файл README.md з детальними інструкціями щодо запуску, скріншотами результатів та вихідним кодом.
