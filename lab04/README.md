
## Опис програми

Цей проект є програмною реалізацією спрощеної системи цифрових підписів на Python. Він створений для демонстрації ключових принципів хешування, асиметричних ключів та перевірки цілісності даних, як того вимагає технічне завдання

Виконання функціональних вимог
Програма повністю реалізує всі обов'язкові вимоги  з технічного завдання.

1. Генерація пари ключів на основі персональних даних

Приватний ключ генерується функцією derive_private_key. Вона створює унікальний рядок з введених даних, кодує його в байти та обчислює SHA-256 хеш, який і слугує приватним ключем .

Публічний ключ обчислюється функцією public_key_from_private. Вона бере байти приватного ключа, перетворює їх на велике число та застосовує спрощену математичну формулу (приватний_ключ * 7) mod 1_000_007, як зазначено у прикладі .

### Метод 1: Через Вбудований Термінал VSC 

Це найкращий спосіб для взаємодії з програмою (введення пароля, імені та дати народження).

1.  У Visual Studio Code, відкрийте термінал, натиснувши **`Ctrl` + `` ` ``** (клавіша з тильдою) або перейдіть у меню **Terminal** $\rightarrow$ **New Terminal**.
2.  Переконайтеся, що ви знаходитесь у каталозі `lab01`. Якщо ні, перейдіть туди:
    ```bash
    cd lab01
    ```
3.  Запустіть програму командою:
    ```bash
    python lab02saltevska.py
    ```
    Після цього програма запитає у вас необхідні вхідні дані.

### Метод 2: Через кнопку "Запуск" (Run)

1.  Переконайтеся, що файл **`lab02saltevska.py`** відкрито та активний.
2.  Натисніть на **трикутник (Play/Run)** у правому верхньому куті вікна VSC.
3.  Програма запуститься, і взаємодія з нею (введення даних) відбуватиметься у панелі **TERMINAL** внизу екрана.
