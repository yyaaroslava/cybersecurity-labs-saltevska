
## Опис програми

Ця програма реалізує базову LSB-стеганографію (Least Significant Bit) на Python для приховування текстових повідомлень у зображеннях формату PNG. Вона дозволяє:

Приховувати текст у пікселях зображення.

Витягувати приховані повідомлення з модифікованого зображення.

Проводити базовий аналіз використання бітів і візуальної непомітності змін.

Програма не використовує сторонні бібліотеки стеганографії — логіка приховування даних реалізована власноруч.

Для запуску цієї програми необхідний встановлений інтерпретатор **Python 3**.


Функціональні можливості

hide_message(cover_path, stego_path, message_text)

Приховує текст у зображенні (cover_path) і зберігає нове зображення як stego_path.

Додає спеціальний маркер кінця повідомлення (END_MARKER) для правильного витягування.

Виводить статистику використаних бітів і співвідношення BPP (бит на піксель).

extract_message(stego_path)

Витягує прихований текст із стего-зображення.

Перевіряє наявність маркера кінця повідомлення.

Виводить витягнутий текст у консоль.

### Метод 1: Через Вбудований Термінал VSC 

Це найкращий спосіб для взаємодії з програмою (введення пароля, імені та дати народження).

1.  У Visual Studio Code, відкрийте термінал, натиснувши **`Ctrl` + `` ` ``** (клавіша з тильдою) або перейдіть у меню **Terminal** $\rightarrow$ **New Terminal**.
2.  Переконайтеся, що ви знаходитесь у каталозі `lab01`. Якщо ні, перейдіть туди:
    ```bash
    cd lab01
    ```
3.  Запустіть програму командою:
    ```bash
    python lab02saltevska.py
    ```
    Після цього програма запитає у вас необхідні вхідні дані.

### Метод 2: Через кнопку "Запуск" (Run)

1.  Переконайтеся, що файл **`lab02saltevska.py`** відкрито та активний.
2.  Натисніть на **трикутник (Play/Run)** у правому верхньому куті вікна VSC.
3.  Програма запуститься, і взаємодія з нею (введення даних) відбуватиметься у панелі **TERMINAL** внизу екрана.
